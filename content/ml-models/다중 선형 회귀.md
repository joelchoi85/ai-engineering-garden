[[선형 회귀]]의 한계(특성 하나)를 푼 것. 현실 문제는 결과를 정하는 요인이 여럿이다(집값 = 방 개수 + 위치 + 연식 + 평수…).

$$\hat{y} = w_1 x_1 + w_2 x_2 + \dots + w_n x_n + b = Xw + b$$

여기서 [[행렬곱]] $Xw$가 등장한다 — 데이터 행렬 × 가중치 벡터. **이 식이 곧 신경망 한 층 [[nn.Linear]]의 정체**다. 선형대수([[행렬]]·[[행렬곱]]) → 회귀 → 신경망이 같은 식으로 한 줄에 꿰인다.

주의: 특성마다 스케일(단위)이 다르면 계수의 크기만으로 중요도를 단정할 수 없다 → [[피처 스케일링]]이 필요한 이유. 평가는 [[모델 평가]]의 R²·RMSE 등으로 한다.
