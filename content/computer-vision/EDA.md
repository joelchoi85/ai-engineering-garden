# EDA란

본격적인 분석이나 머신러닝 모델링 전, **데이터를 다양한 각도에서 관찰하고 핵심 특성을 파악하는 필수 과정**

## 💡 EDA 핵심 5단계 수행 방법

1. **데이터 형태 및 구조 파악 (Structure & Shape)**
    - 데이터의 행(row)과 열(column) 개수, 각 변수의 데이터 타입(수치형, 범주형)을 확인.
    - 결측치(비어있는 값)가 얼마나 있는지 확인하고 처리 계획 세움.
2. **기초 통계량 확인 (Summary Statistics)**
    - 데이터의 요약 통계량(평균, 중앙값, 최솟값, 최댓값, 표준편차 등)을 계산해 전체적인 데이터 범위 파악.
3. **단일 변수 분석 (Univariate Analysis)**
    - 하나의 변수만을 독립적으로 관찰하여 데이터의 분포 상태 확인.
    - 숫자형 데이터는 히스토그램이나 상자 그림(Box plot), 범주형 데이터는 막대그래프를 주로 사용.
    - 일반적인 범위를 크게 벗어나는 **이상치(Outlier)** 를 발견하고 제거할지 판단.
4. **다변수 간의 관계 분석 (Bivariate Analysis)**
    - 두 개 이상의 변수 간에 어떤 관계가 있는지(상관관계) 파악.
    - 산점도(Scatter plot)나 상관계수 행렬 활용.
    - ex) '광고비'와 '매출' 사이에 양의 상관관계가 있는지 확인하는 작업.
5. **가설 검정 및 심층 분석**
    - 앞선 단계에서 발견한 패턴이나 의문점을 바탕으로 가설을 세우고 이를 검증.

# 이미지 분석에서의 EDA (탐색적 데이터 분석)

수치형 데이터와 달리 눈으로 직접 데이터를 확인하고, 이미지의 컴퓨터 표현 방식(픽셀, 채널, 메타데이터)을 분석함.

> 모델 학습 전 데이터의 오염, 불균형, 다양성을 파악하는 것이 핵심
---
## 🖼️ 이미지 EDA 핵심 5단계 수행 방법

### 1. 메타데이터 파악 및 정렬 (Metadata Check)
* **파일 확장자:** `.jpg`, `.png` 등이 통일되었는지 확인.
* **해상도 분포:** 각 이미지의 가로/세로 크기가 제각각인지 분포 파악.
* **채널 수:** RGB(3채널), Grayscale(1채널), RGBA(4채널) 구조 확인.

### 2. 클래스 불균형 확인 (Class Imbalance)
* **데이터 분포:** 분류(Classification) 문제의 경우, 각 카테고리별 이미지 개수가 균등한지 파악.
* **증강 대상 선정:** 데이터 수가 너무 적은 클래스는 향후 [[데이터 증강(Augmentation)]] 대상으로 지정.

### 3. 샘플 시각화 및 무작위 검수 (Visual Inspection)
* **무작위 출력:** 각 클래스별로 이미지를 무작위로 여러 장 출력하여 눈으로 검수.
* **데이터 오염 탐지:** 이미지가 깨졌거나, 타겟 물체가 잘렸거나, 글씨/워터마크가 박혀있는지 확인.

### 4. 픽셀 값 분포 분석 (Pixel Intensity Analysis)
* **통계량 확인:** 이미지 전체 또는 채널별 픽셀 값(0~255)의 평균, 표준편차, 히스토그램을 분석.
* **조도 파악:** 특정 이미지들이 너무 어둡거나(저조도) 너무 밝은지(과노출) 파악하여 [[정규화(Normalization)]] 전략 세움.

### 5. 객체 탐지/분할 데이터 검수 (Annotation Check)
* **라벨 일치성:** 바운딩 박스(Bounding Box)나 분할(Segmentation) 마스크가 정답 위치와 일치하는지 이미지 위에 겹쳐서(Overlay) 시각화.
* **객체 크기 분포:** 물체의 크기(예: 너무 작은 객체 비율)와 위치 분포 파악.

## 🛠️ 이미지 EDA 코드 예시 (Python/OpenCV)

> [!tip] 팁
> 아래 코드를 실행하여 이미지의 기초 정보를 빠르게 확인할 수 있음.

```python
import cv2
import matplotlib.pyplot as plt

# 1. 이미지 로드 및 기본 정보 확인
img = cv2.imread('sample.jpg')
h, w, c = img.shape
print(f"해상도: {w}x{h}, 채널 수: {c}")

# 2. 픽셀 통계량 확인
print(f"픽셀 최솟값: {img.min()}, 최댓값: {img.max()}, 평균: {img.mean():.2f}")

# 3. 이미지 시각화
plt.imshow(cv2.cvtColor(img, cv2.COLOR_BGR2RGB))
plt.axis('off')
plt.show()
```
